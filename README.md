# Level Controll

A frustratingly addictive 2D platformer inspired by "Level Devil". Features 10 levels of increasing difficulty, deceptive traps, and aggressive "troll" mechanics designed to test your patience.

- **10 Deceptive Levels**: Moving spikes, fake blocks, invisible walls, and more.
- **Aggressive Trolling**: real-time mocking popups and goal-fleeing mechanics.
- **The Ultimate Troll**: Excessive failure (20+ deaths) on Level 1 leads to an external "redemption" video and a temporary ban.
- **Secret Redemption**: A hidden way to regain access if you're persistent enough.
- **Synthesized Audio**: Dynamic sound effects generated via Web Audio API (no external assets needed).
- **Responsive Controls**: Desktop keyboard support and touch controls for mobile.
- **Persistence**: Level unlocks and fail counts are saved to LocalStorage.
